Exporter le code HTML

Pour conserver vos modifications de façon permanente, copiez ce code HTML complet et remplacez le contenu de votre fichier sur GitHub ou votre hébergeur web.

Comment utiliser ce code :

  1. Copiez tout le code HTML en cliquant sur le bouton "Copier le code"
  2. Allez sur votre dépôt GitHub où le site est hébergé
  3. Trouvez le fichier HTML principal (généralement index.html)
  4. Cliquez sur le bouton "Modifier" (icône de crayon)
  5. Supprimez tout le contenu actuel
  6. Collez le nouveau code HTML
  7. Validez les modifications avec un message comme "Mise à jour du portfolio"
